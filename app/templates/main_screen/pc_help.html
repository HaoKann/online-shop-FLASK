Видеокарта — это компонент компьютера, отвечающий за обработку и вывод изображения на экран. Она рендерит графику в играх, обрабатывает видео, помогает при работе с 3D, графическим дизайном и ускоряет вычисления в программах (например, монтаже или ИИ).

Есть два типа видеокарт:

  • 1. Встроенная (интегрированная) — находится в процессоре, подходит для базовых задач: браузер, видео, офис.
  • 2. Дискретная (отдельная) — мощный отдельный компонент, нужен для игр, графики, монтажа и т.д.

Как подобрать видеокарту?

  1. Цель использования: Для интернета и фильмов мощная видеокарта не нужна. Для игр или монтажа — дискретная.
  2. Разрешение экрана: Выше разрешение (Full HD, 4K) требует мощности для плавности.
  3. Объем VRAM: 6–8 ГБ для Full HD, 12+ ГБ для 4K или тяжелых задач.
  4. Совместимость: Проверка блока питания и размера корпуса.
  5. Тесты: Смотрите FPS в обзорах.

Процессор (CPU) — "мозг" компьютера, выполняющий все вычисления и команды.

Основные характеристики: количество ядер, потоков, частота (ГГц) и архитектура.

Как подобрать процессор?

  1. Цель: Intel i3/Ryzen 3 для офиса, i5/Ryzen 5 для игр, i7/Ryzen 7 для монтажа.
  2. Ядра и потоки: 4 ядра — минимум, 6–8 — для игр, 12+ — для профессиональных задач.
  3. Частота: Выше ГГц — лучше для игр.
  4. Совместимость: Подходит ли сокет (AM4, LGA1700).
Выбрать процессор

Материнская плата — основа ПК, соединяющая все компоненты.

Как подобрать?

  1. Сокет: Совместим с процессором (LGA1700, AM4).
  2. Форм-фактор: ATX для больших сборок, Mini-ITX для компактных.
  3. Чипсет: Z690 для разгона, B/H для базовых задач.
  4. Охлаждение: Поддержка вентиляторов и VRM.
  5. Порты: USB, M.2, Wi-Fi.
Выбрать материнскую плату

Блок питания преобразует ток для работы ПК и стабилизирует напряжение.

Как подобрать?

  1. Мощность: 450–650 Вт для базовых ПК, 750+ Вт для мощных сборок.
  2. КПД: 80 PLUS Gold/Bronze.
  3. Разъемы: 24-pin, 8-pin CPU, 6+2-pin GPU.
  4. Защита: От перегрузки и перегрева.
Выбрать блок питания

Оперативная память (ОЗУ) — временное хранилище для текущих задач.

Как выбрать?

  1. Объем: 8 ГБ для офиса, 16 ГБ для игр, 32+ ГБ для монтажа.
  2. Стандарт: DDR4 дешевле, DDR5 быстрее.
  3. Частота: 2666–3600 МГц для DDR4, 4800+ МГц для DDR5.
  4. Тайминги: Меньше CL — выше скорость.
  5. Канальность: 2×8 ГБ лучше 1×16 ГБ.
Выбрать ОЗУ

Кулеры и системы охлаждения отводят тепло от компонентов.

Как выбрать?

  1. TDP: Средний кулер для i5, СВО для i9.
  2. Совместимость: Подходит ли сокет и корпус.
  3. Шум: Выбирайте модели с низким уровнем дБ.
  4. Бренд: Noctua, be quiet!, DeepCool.
  5. RGB: По желанию для внешнего вида.
Выбрать кулер

Жёсткие диски (HDD) и SSD — устройства хранения данных.

Как выбрать?

  1. Цель: HDD для хранения, SSD для скорости.
  2. Скорость: NVMe SSD для игр и видео.
  3. Ёмкость: HDD для архивов, SSD для ОС.
  4. Надёжность: SSD устойчивее к поломкам.
Выбрать накопитель

Корпус — оболочка для компонентов ПК.

Как выбрать?

  1. Форм-фактор: ATX для больших, Mini-ITX для компактных.
  2. Охлаждение: Поддержка вентиляторов и СВО.
  3. Порты: USB, аудио на передней панели.
  4. Кабели: Удобство управления проводами.
  5. Размер: Подходит ли для видеокарты.
